## Requirements

* Python 3.8 o superior
* virtualeenv
* Conexión a Watson DB (podemos_utils)


## Instalar proyecto

```sh
make install
```

## Levantar proyecto

```sh
make up
```

## linter al proyecto

```sh
make lint
```


## linter fix al proyecto

```sh
make lint-fix
```


## Ejecutar tests

```sh
make test
```
